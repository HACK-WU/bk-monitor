# Issue 详情页设计稿

## 概述
Issue 详情页用于展示单个 Issue 的完整信息，包括告警趋势、维度统计、基本信息、问题活动历史等。

---

## 1. 页面头部 (Title Header)

### 1.1 整体布局

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  ◀   [回归]  NulpointerException ▼  异常登录日志告警         ⟨ 近 7 天 ⟩  🔄  ⬜  ⛶ │
└─────────────────────────────────────────────────────────────────────────────┘
```

**布局说明**:
- **返回按钮**: 左侧箭头图标 `◀`，点击返回 Issue 列表
- **回归标记**: 黄色/橙色标签 `[回归]`
- **标题区域**:
  - 主标题: `NulpointerException` (带展开/折叠图标 `▼`)
  - 副标题: `异常登录日志告警`
- **右侧操作区**:
  - 时间选择器: `⟨ 近 7 天 ⟩` (左右箭头切换)
  - 刷新按钮 `🔄` (圆形箭头图标)
  - 新开页按钮 `⬜`
  - 全屏按钮 `⛶`

### 1.2 状态标记

```
回归状态:  [回归] NulpointerException ▼  异常登录日志告警
           ════
           橙黄色标签

新状态:    [新] NulpointerException ▼  异常登录日志告警
           ═══
           绿色标签
```

- **回归状态**: 橙色标签，表示 Issue 重新出现
- **新状态**: 绿色标签，表示新创建的 Issue

## 2. 检索栏 (Search Bar)

### 2.1 功能说明

检索栏用于 **过滤当前 Issue 关联的告警列表**，帮助用户在大量关联告警中快速定位关键信息。

| 维度 | 说明 |
|------|------|
| **搜索范围** | 单个 Issue 关联的告警子集（`AlertDocument` 中 `issue_id` 等于当前 Issue ID 的告警） |
| **不是** | 搜索整个告警中心的所有告警 |
| **使用场景** | 下钻分析特定问题的告警详情 |

**搜索场景示例**：

假设当前 Issue 是 "主机 CPU 使用率过高"，关联了 86 条告警，用户可以在检索栏：
- 按主机 IP 过滤：只看特定主机的告警
- 按告警状态过滤：只看未恢复/已恢复/已失效的告警
- 按时间过滤：只看某个时间段内的告警
- 关键词搜索：在告警内容中搜索关键词

**与告警中心搜索的区别**：

| 维度 | 告警中心 | Issue 详情页 |
|------|----------|--------------|
| 数据范围 | 全业务/全策略的所有告警 | 单个 Issue 关联的告警 |
| 过滤条件 | 无 `issue_id` 限制 | 固定 `issue_id = current_issue.id` |
| 使用场景 | 发现新问题 | 下钻分析特定问题的告警详情 |

### 2.2 交互规范
- 跟 `<告警中心>` 的检索栏交互保持一致
- 为了和页面风格保持一致，圆角改为 8px，增加 border

### 2.3 组件布局

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  UI模式 ▼  + 添加条件    唤起，输入检索内容...                      ⚙  ☆  🔍  │
│  ════════  ═══════════  ════════════════════════════════════════  ═  ═  ══  │
│   下拉选择   按钮          输入框(占位符)                            筛  收  搜 │
│                                                                   选  藏  索 │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2.3 组件内容
- **UI 模式切换**: 下拉选择 `UI模式 ▼`
- **添加条件按钮**: `+ 添加条件`
- **搜索输入框**: 占位符 `唤起，输入检索内容`
- **快捷操作图标**:
  - 筛选图标 `⚙`
  - 收藏图标 `☆`
- **搜索按钮**: 蓝色主按钮 `🔍` 圆形图标按钮

## 10. 交互说明

### 10.1 时间选择器
- 左右箭头切换时间范围
- 点击时间文本可展开更多选项

### 10.2 趋势图表
- 悬停显示具体数值
- 点击可下钻查看详情

### 10.3 维度统计
- 悬停显示详细占比和具体值
- 点击可筛选相关告警

### 10.4 评论功能
- 最多显示 3 行
- 超出显示 `...` 和展开按钮
- 点击展开按钮弹出完整内容弹窗
